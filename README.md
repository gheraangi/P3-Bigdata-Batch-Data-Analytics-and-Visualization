# Praktikum Big Data Analytics

## Deskripsi Project
Project ini merupakan implementasi sederhana dari pipeline Big Data yang terdiri dari beberapa tahapan, yaitu data ingestion, data processing, dan analytics layer. Data yang telah diproses kemudian digunakan untuk membuat visualisasi dashboard menggunakan Power BI.

Hasil analisis menghasilkan beberapa informasi utama seperti:
- Total revenue
- Produk dengan penjualan terbanyak
- Revenue berdasarkan kategori produk

Output dari proses analytics disimpan dalam folder `data/serving` dan digunakan untuk visualisasi dashboard di Power BI.
